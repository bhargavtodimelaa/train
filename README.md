> ⚡ A modern, fast, and elegant **Live INDIAN Train Tracking Web App** with real-time updates, smart search, and network insights.
> DIRECT LINK- https://train-psi-rouge.vercel.app/

## ✨ Features

- 🔍 Smart Indian  Train Search  
- 🚆 Live Train Tracking (Auto-refresh 45s)  
- 🗺️ Map Integration (Leaflet.js)  
- 🌐 Network Monitor (IP, ISP, latency, signal)  
- ⭐ Favourites System  
- 🌙 Dark / Light Mode  
- 📡 Offline Detection  
- 📱 Fully Responsive UI
- ALL STOPS DATA WITH EXPECTED DELAY ARRIVAL TIMES AND PLATFORM NUMBERS DATA
- SHOWS LIVE LOCATION OF TRAIN WITH EXACT COORINATES ON MAP(SUPPORTS OPEN MAP AND GOOGLE MAPS(EMBEDDED)
- IN BETA STAGE 

---

## 🧱 Tech Stack

- HTML5, CSS3, JavaScript (Vanilla)  
- Leaflet.js (Maps)  
- Google Fonts  
- Unofficial Train APIs + Cloudflare Worker Proxy

🧠 How It Works

User Input → API Fetch → Process Data → UI Render → Map Update → Auto Refresh
